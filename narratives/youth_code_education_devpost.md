# CogPace — Youth Code × AI (Track 04: Career / Education)

**Run:** `COGPACE_PROFILE=youth_code_education streamlit run app.py`

## Pitch

Career tools ask "what do you like?" CogPace asks **how you learn**. Your attention profile (% OPTIMAL vs OVERLOADED per subject) is a cognitive fingerprint for exploring STEM paths.

## Features for this track

- Session summary → attention profile breakdown
- Cerome-lite labels: hyperfocus-prone / load-sensitive / underload-prone (emergent, not MBTI)
- AI career hints linked to subject + engagement pattern
- Follow-up chat: "What careers use this topic?"

## Science

MFA (Zeng, 2026): F_att(r)=S/r² unifies five attention theories. ADHD-hyperfocus continuum modeled by σ (Castellanos & Tannock, 2002).
